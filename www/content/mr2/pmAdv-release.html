phoneME Advanced MR2 Release Notes

Features

phoneME Advanced MR2 software is an optimized Java ME stack with full support for CDC and its profiles.

  • Connected Device Configuration, version 1.1.2

  • Foundation Profile, version 1.1.2

  • Personal Basis Profile, version 1.1.2

phoneME Advanced MR2 is in the early stages of development. This development release of MR2 represents the basic foundation from which the project will evolve. For more information on the expected feature set and direction of phoneME Advanced technology, see the phoneME Advanced MR2 Software Release Description.

Binary Archives

As a convenience, phoneME Advanced releases provide prebuilt binary Zip archives in the phoneME Project Downloads page. The distribution archive actually contains two sub-archives, each of which contains an executable CDC Java runtime environment for Linux/x86. 

% mkdir phoneme-advanced-rel
% cd phoneme-advanced-rel
% unzip phoneme_advanced-rel-bin-b06-linux-date.zip
Archive:  phoneme_advanced-rel-bin-b06-linux-date.zip
 ...

Then unpack one of these archives: 

% mkdir PBP
% mv phoneme_advanced-rel-pbp-bin-b06-linux-date.zip PBP
% cd PBP
% unzip phoneme_advanced-rel-pbp-bin-b06-linux-date.zip PBP

See CDC Build System: Host Environment and the CDC Runtime Guide, for information how to use the CDC Java runtime environment.

Known Issues

The font path may need to be explicitly set on openSUSE 10.1 systems. To check the font path on the openSUSE 10.1 system, use the following command: 

% xset q

To able to display correctly on an openSUSE 10.1 system the font path should at least contain the following: 

/usr/X11R6/lib/X11/fonts/misc:unscaled
/usr/X11R6/lib/X11/fonts/75dpi
/usr/X11R6/lib/X11/fonts/uni
/usr/X11R6/lib/X11/fonts/truetype/

To set the font path 

% xset +fp
