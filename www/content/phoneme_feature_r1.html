Phone ME Feature MR1

Release Description

PhoneME-feature-MR1 is an implementation of the Java™ Platform, Micro Edition (Java ME platform) based on the Connected Limited Device Configuration (CLDC), HotSpot™ Implementation, version 1.1.3. The release includes an implementation of the MIDP 2.0.1 specification, which is extended to support multitasking. The multitasking is layered on a single OS process with each MIDlet running in its own logical virtual machine that is independent of the other MIDlets.

Feature List

  • The following subset of MSA 248 optional packages:
    • JSR 75 (PIM and File management)
    • JSR 82
    • JSR 135
    • JSR 172
    • JSR 177
    • JSR 184
    • JSR 205
    • JSR 226
  • Multitasking capabilities supporting both multiple and simultaneous MIDlets

Platforms Supported

The supported platforms are Linux/ARM and Windows/i386.

The platform configurations are:

  • TI P2SAMPLE64-V6 with Linux/ARM
  • Windows/i386 (in Emulation mode)
