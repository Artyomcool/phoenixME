phoneME Advanced MR2

Release Description

phoneME Advanced MR2 represents the next major milestone release of phoneME Advanced. While MR1 was developed behind closed doors, MR2 represents the first release of phoneME Advanced that will be developed with the participation of the Mobile & Embedded community.

phoneME Advanced MR2 development activity is expected to start within the next couple of months.

Feature List

While the full feature set associated with phoneME Advanced MR2 is still being finalized, the primary release objectives associated with phoneME Advanced MR2 include:

  1. Full MIDP Support (compatible with MIDP on phoneME Feature MR2)

  2. Full or partial implementation of the following MSA JSRs:

    • Personal Information and File Management (JSR 75)

    • Bluetooth and OBEX (JSR 82)

    • Wireless Messaging API (JSR 120)

    • Mobile Media API (JSR 135)

    • 3D Graphics (JSR 184)

    • Wireless Messaging API 2.0 (JSR 205)

    • Scalable 2D Vector Graphics (JSR 226)

Additional release requirements and criteria will be added prior to initiating the phoneME Advanced MR2 project. If you have any requirements that you would like considered for inclusion in MR2, please email us and/or participate!

Note: some of the requirements above may change and while we will do our best to accommodate requests, we may not be able to accommodate everything.
